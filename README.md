# Revit BIM Automation Toolkit

Набор инструментов для автоматизации рутинных задач в Autodesk Revit, разработанный в рамках реальных проектов промышленного и гражданского строительства (в т.ч. в контуре структур «Ростех»).

## О проекте

Проект решает проблему ручной координации разделов и проверки моделей. Вместо часового кликанья мышкой — запуск скрипта и минутное ожидание результата. Все инструменты «выросли» из личной практики BIM-координатора разделов АР и КЖ, который устал делать одно и то же руками.

## Состав и возможности

- **Пакетная обработка файлов:** Асинхронная работа с десятками файлов Revit без открытия интерфейса.
- **Автоматическая координация:** Программное связывание файлов (`RevitLinkType.Create`), захват координат (`AcquireCoordinates`), управление позициями проектов.
- **Валидация и проверки:** Автоматизированная проверка моделей на соответствие BEP (План реализации BIM-проекта). Поиск и логирование коллизий.
- **Подавление предупреждений:** Реализован продвинутый `IFailuresPreprocessor` для бесшовной работы без остановок на диалоговые окна Revit.
- **Логирование и отчёты:** Детальное журналирование каждого шага + генерация финального отчёта о проделанной работе. Полная прозрачность для пользователя.
- **Кастомный UI:** Приложение с графическим интерфейсом на Windows Forms для удобной работы непрограммистов.

## Технологический стек

- **Язык:** C# (.NET Framework / .NET 8+)
- **Платформа:** Autodesk Revit API
- **Интерфейс:** Windows Forms (WinForms)
- **Управление кодом:** Git
- **Окружение:** Windows, Revit 2022+

## Пример использования (псевдокод)

// 1. Пакетная обработка связей
var linkOptions = new RevitLinkOptions(true);
using (var t = new Transaction(doc, "Auto-Link"))
{
    t.Start();
    RevitLinkType.Create(doc, path, linkOptions);
    t.Commit();
}

// 2. Бесшовная обработка предупреждений
var processor = new WarningSwallower(); // реализует IFailuresPreprocessor
var options = new SaveOptions();
options.SetFailuresPreprocessor(processor);
document.Save(options);
// Всё сохранилось без единого диалогового окна
